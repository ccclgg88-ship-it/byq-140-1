## 三、开发实现阶段

### 3.1 按待办清单逐步实现

- 用户要求：根据开发待办清单一步一步实现项目。
- **执行与产出**：
  - **阶段一**：创建目录结构（css/、js/、images/、images/works/、images/icons/），5 个 HTML 骨架，CDN 引入（Bootstrap、jQuery、jQuery UI、jQuery Mobile），统一导航片段。
  - **阶段二**：编写 `css/common.css`（设计令牌、基础重置、导航、标题正文、通用组件、页脚、响应式）。
  - **阶段三**：编写 `js/common.js`（当前页导航高亮、showToast、localStorage 工具、表单基础校验）。
  - **阶段四**：封面页完整内容与 `cover.css`（非对称布局、职业照、技能标签、联系方式）。
  - **阶段五～八**：个人定位页、技能与工具页、作品与项目页、教育与总结页的 HTML 与各自 CSS，内容按需求填写完整。
  - **阶段九**：占位图（`images/avatar.svg`、`images/works/placeholder.svg`），封面与作品图缺图时回退逻辑。
  - **阶段十**：编写 `README.md`，开发待办清单全部勾选并更新修订历史。

### 3.2 问题修复：浏览器反复刷新与 avatar.svg 重复请求

- **现象**：用浏览器打开 index.html 后浏览器“不同断刷新”，Network 显示 `avatar.svg` 被请求数百次（如 474 次），均由 index.html 第 37 行触发。
- **原因**：① 图片 onerror 在 fallback 到 avatar.svg 后，若 SVG 加载失败会再次触发 onerror 形成循环；② jQuery Mobile 的 AJAX 导航与 hash 监听在多页静态场景下导致重复加载/增强。
- **处理**：
  - 为头像与作品图 onerror 增加 `data-fallback` 判断，仅执行一次 fallback，避免循环请求。
  - 在所有页面引入 jQuery Mobile 前通过 `mobileinit` 关闭 `ajaxEnabled`、`linkBindingEnabled`、`hashListeningEnabled`。
- **产出**：修改 `index.html`、`works.html` 及 5 个页面的 script 片段。

## 四、体验与视觉优化阶段

### 4.1 美观度提升至「现代官网级别」

- 用户要求：在需求的设计与美观度基础上，增加布局美观、层次清晰、交互自然、现代简洁、留白与配色、圆角阴影、hover/点击反馈、语义化与响应式、加载与交互流畅。
- **执行与产出**：
  - 重写 `css/common.css`：新设计令牌（主色/辅助色/中性色、间距尺度、圆角与阴影层级、过渡变量）、渐变背景、导航与按钮/卡片/表单的 hover 与 focus、`.tech-highlight` 样式。
  - 更新各页专属 CSS（cover、positioning、skills、works、education）：留白、层次、卡片与列表 hover、技能标签与进度条、作品卡与图片 hover 动效、求职意愿块样式。
  - 保持语义化与响应式，无横向滚动与重叠。

### 4.2 技能图标：从 Unicode 到 Bootstrap Icons

- **上下文**：用户查看技能图标的实现方式，当时为 Unicode 字符「◆」+ CSS。
- 用户要求：用图标字体或 SVG 图标库（如 Bootstrap Icons）替换 ◆ 的 HTML/CSS。
- **执行与产出**：
  - 在 5 个页面引入 Bootstrap Icons CDN。
  - 封面技能标签：PS/AI → `bi-palette-fill`，C4D → `bi-box-fill`，Figma → `bi-grid-3x3-gap-fill`，HTML/CSS → `bi-code-slash`。
  - 技能页：设计/技术工具项使用对应图标（palette-fill、box-fill、grid-3x3-gap-fill、code-slash、filetype-py、file-earmark-spreadsheet-fill）；能力列表改为 `bi-check2-circle-fill`。
  - 更新 `cover.css`、`skills.css` 中图标类名与样式，移除原 `::before` 菱形；为图标添加 `aria-hidden="true"`。

### 4.3 C4D 图标不显示

- **现象**：技能页 C4D 项旁 `bi-cube-fill` 图标未显示。
- **原因**：Bootstrap Icons 无 `bi-cube` / `bi-cube-fill`，仅有「盒子/立方」类图标 `bi-box`、`bi-box-fill`。
- **处理**：将 C4D 图标由 `bi-cube-fill` 改为 `bi-box-fill`（index.html、skills.html）。

## 七、关键决策与约束

- **架构**：多页静态 + 公共资源，无构建、无后端，与需求「纯前端、简单可维护」一致。
- **技术栈**：严格按需求使用 HTML5、CSS3、JavaScript、Bootstrap、jQuery、jQuery UI、jQuery Mobile，并增加 Bootstrap Icons；未引入 React/Vue 等框架。
- **兼容与稳定性**：jQuery Mobile 采用 1.4.5 可用 CDN；通过 mobileinit 关闭 AJAX 与 hash，避免与多页静态冲突；图片加载失败有占位与防循环。
- **文档链**：需求说明 → 开发设计 → 开发待办 → 实现 → 图片提示词 / README / 完成情况说明，形成可追溯的轨迹。
