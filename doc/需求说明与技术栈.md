## 需求说明

1.网站概述
1.1 网站名称
个人简历展示网站
1.2 网站描述
面向高职院校“广告艺术设计”专业毕业生的个人简历展示静态网站，突出「设计+技术」双维度能力，为求职提供专业的线上展示平台。（要求至少包含5个独立静态HTML网页，含CSS样式表、JS文件、图片文件夹等）
2.技术要求
2.1 前端技术栈
·  HTML5 ：语义化标签、表单控件、媒体元素
·  CSS3 ：Flex/Grid布局、动画、过渡、响应式设计
·  JavaScript ：DOM操作、事件监听、表单验证、本地存储（localStorage）
·  Bootstrap ：响应式栅格系统、组件（导航栏、卡片、轮播图）
·  jQuery UI ：交互组件（日期选择器、对话框）
·  jQuery Mobile ：移动端适配（优化触屏体验）
2.2 文件结构
· 多个独立静态HTML页面
· 单独的CSS样式表
· 图片资源文件夹
· 媒体文件夹
3.页面功能模块
3.1 封面页（视觉+定位）
3.1.1 核心目标
用设计感快速抓住HR注意力，明确求职方向
3.1.2 内容结构
·  视觉元素 ：个人职业照（简洁背景，体现设计审美）
·  创意标题 ：「广告设计×技术落地 | 用设计解决问题的实践者」
·  关键信息 ：姓名、求职岗位（广告设计师/视觉设计师/UI设计师）
·  核心技能标签 ：PS/AI、C4D、Figma、HTML/CSS（图标+文字组合）
·  联系方式 ：电话、邮箱、作品集链接（Behance/个人网站）
3.1.3 设计重点
· 配色方案：高饱和度对比色（蓝+橙、黑+白），体现专业感
· 排版布局：非对称布局（左侧照片+右侧文字），避免呆板感
· 整体风格：广告设计常用配色，突出专业性和创意感
3.2 个人定位页（专业+技术结合）
3.2.1 核心目标
用一句话定义核心价值，突出「设计+技术」差异化优势
3.2.2 内容结构
·  个人简介 ：高职院校计算机信息学院广告设计专业毕业生，既懂设计创意，又能通过技术实现落地
·  职业理念 ：以用户需求为核心，用设计解决商业问题，用技术提升设计效率
·  技术赋能案例 ：
o Python批量处理设计素材，效率提升50%
o Figma做交互式原型，让客户提前体验设计效果
3.2.3 设计重点
· 代码块样式突出技术关键词（Python、Figma）
· 设计元素分隔内容，视觉上区分「设计」与「技术」模块
· 左右分栏布局，平衡「设计」与「技术」比重
3.3 技能与工具页（设计+技术双维度）
3.3.1 核心目标
清晰展示「设计技能」与「计算机信息能力」的具体内容，证明双专业优势
3.3.2 内容结构
·  设计技能模块 ：
o 软件操作：Photoshop（精通）、Illustrator（精通）、C4D（熟练）、Figma（熟练）
o 设计能力：平面设计、UI设计、品牌视觉、海报设计（案例支撑）
·  计算机信息能力模块 ：
o 技术工具：HTML/CSS（基础）、Python（基础，用于设计自动化）、Excel（数据分析）
o 技术应用：网页设计、设计素材自动化处理、用户数据可视化
3.3.3 设计重点
· 左右分栏布局（左侧设计技能，右侧技术能力）
· 每个技能配小图标（PS图标、代码图标）增强可读性
· 星级/进度条表示熟练度，直观展示技能水平
3.4 作品与项目页（核心页，设计+技术落地）
3.4.1 核心目标
用具体项目证明「设计+技术」结合能力，突出实战经验
3.4.2 内容结构
·  项目1：广告设计类
o 项目名称：校园文化节海报设计
o 设计过程：需求分析→创意构思→软件实现（PS/AI）→效果优化（C4D添加3D元素）
o 成果：获学院「最佳创意奖」，印刷量500份，覆盖全校
·  项目2：技术赋能设计类
o 项目名称：交互式产品宣传页设计
o 技术应用：Figma做原型→HTML/CSS实现静态页面→JavaScript添加交互效果
o 成果：页面访问量1000+，客户反馈「交互体验提升设计说服力」
·  项目3：跨专业合作类
o 项目名称：电商平台商品详情页设计
o 合作内容：与计算机专业同学合作，负责视觉设计，配合前端同学用Figma交付规范切图
o 成果：页面转化率提升15%
3.4.3 设计重点
· 大图+文字组合展示（左侧作品图，右侧项目说明）
· 技术关键词用橙色高亮区分（HTML/CSS实现、Figma交付规范）
· 数据化表达成果（效率提升50%、转化率提升15%）
3.5 教育与总结页（专业+技术背景+求职意愿）
3.5.1 核心目标
补充教育背景，强化「设计+技术」专业基础，表达求职诚意
3.5.2 内容结构
·  教育背景 ：
o 院校：高职院校计算机信息学院
o 专业：广告设计
o 核心课程：设计类（平面构成、广告创意）+ 技术类（网页设计、Python基础）
·  求职意愿 ：
o 对岗位理解：广告设计不仅是创意，更是用技术实现商业价值的过程
o 期望：希望加入重视「设计+技术」结合的团队，用设计能力和技术基础，为品牌创造更有影响力的视觉内容
3.5.3 设计重点
· 简洁列表展示课程，避免冗长
· 结尾用大字号加粗突出求职意愿
· 强调「设计+技术」结合的价值观
4.设计规范
4.1 视觉一致性
· 配色方案：主色为广告设计常用蓝色，辅助色为橙色
· 字体设置：标题用黑体，正文用微软雅黑
· 技术标签：用图标+高亮方式突出技术关键词（HTML/CSS、Python）
4.2 数据化表达
· 所有成果用数字支撑（效率提升50%、转化率提升15%）
· 避免空泛描述，用具体数据体现价值
4.3 技术标识
· 在每个页面中突出技术关键词，让HR快速识别技术能力
· 区别于纯艺术设计毕业生，体现复合型人才优势
5.关键设计原则

1. 视觉一致性 ：所有页面使用统一的配色方案、字体和设计语言，体现专业度

1. 数据化表达 ：所有成果用具体数字支撑，避免空泛描述，增强说服力
